# SCRIPT FORMATTING & VISUAL LOCKS

To maintain cross-department autonomy and Zero Hallucinations, all script panels MUST include a Visual Lock constraint for the AI generator.

**Format:**
`PANEL 1`
`[VISUAL_LOCK: 1997 Noir, NO_TEXT, NO_LOGOS, 1 Character Only]`
*Description:* Jonah Creed chops meat at the counter.
*Dialogue:* CREED: "Hand me the salt."
